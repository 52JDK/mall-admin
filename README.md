# mall-admin
小R商城 后台管理系统 的后端项目，基于SpringBoot+Shiro+JWT+Redis+MongoDb+Mysql 进行实现的 主要包括 装修页面、商品管理、订单管理、活动管理、优惠券管理、权限管理等

### 项目预览
（1）在线预览地址：http://admin.52jdk.com

（2）前端代码地址：https://github.com/52JDK/web-admin
